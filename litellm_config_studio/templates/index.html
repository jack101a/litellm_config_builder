Recommended flow

  1. Select providers (e.g., NVIDIA, OpenAI) and plan API keys.
  2. Fetch models directly from providers and enrich metadata.
  3. Choose Hybrid mode: explicit aliases for production, wildcard for experiments.
  4. Enable retries, cooldowns, and health routing.
  5. Run cheap local smoke tests before deployment.

Security & Safety

  • Generated configs use secure os.environ/KEY_NAME.
  • Pasted keys are strictly for in-browser live tests.
  • LAN mode is disabled by default.
  • All generation happens locally on this machine.
